- `page.component` 页面组件
> 页面对将业务组件接后台数据


```mermaid
graph TD;
主题组件-->通用组件;
定位组件-->通用组件
容器组件-->通用组件;
展示组件-->通用组件;
通用组件-->单功能组件;

单功能组件-->组合组件;
组合组件-->业务组件;
业务组件-->页面组件;
页面组件-->应用
```
